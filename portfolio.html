MyIoT Project Manager

Manage your IoT devices and projects efficiently

About

MyIoT Project Manager is a web-based tool designed to help users manage, monitor, and control their Internet of Things (IoT) devices and related projects from a centralized dashboard.

Key Features

  • Real-time device monitoring
  • Project and task tracking
  • Automated alerts and notifications
  • Secure authentication system
  • Data visualization dashboards

Technologies Used

  • HTML, CSS, JavaScript (Frontend)
  • Node.js / Python (Backend)
  • MQTT / HTTP Protocols
  • MongoDB / Firebase (Database)
